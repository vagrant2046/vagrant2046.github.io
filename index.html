hexo + github page 部署博客

2020-03-22
技术

1.部署博客前的准备工作

  1. 安装 Node.js
  2. 安装 Git
  3. 注册 Github

2.Hexo的安装及初始化

此时你可以通过 localhost:4000 访问本地的Hexo博客了

3.发布一篇博客到Github上

  1. Github创建一个新的Repository，Repository name 请填写Owner一样的名字，勾选 Initialize this repository with a README 其他默认即可
  1. 获取Repository的Clone链接如：https://github.com/vagrant2046/vagrant2046.github.io.git

  1. 打开 blog/_config.yml 文件配置

  2. 安装 hexo-deployer-git

  3. 选择自己喜欢的主题 以 ayer 主题为例，下载并配置好主题

  4. 生产部署
